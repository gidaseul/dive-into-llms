#LLM 수학적 추론
이 프로젝트는 수학적 추론을 학습하기 위해 대규모 언어 모델(LLM)용 코드를 구현합니다. 주요 기술은 SFT(Supervised Finetuning)를 사용하여 모델이 DeepSeek-R1에서 생성된 응답을 학습하고 복잡한 수학적 추론 기술(예: 반사, 계산 등)을 학습하도록 하는 것입니다.
이 실험은 [DeepMath-103K](https://huggingface.co/datasets/zwhe99/DeepMath-103K) 데이터 세트와 [Qwen/Qwen2.5-Math-1.5B](https://huggingface.co/Qwen/Qwen2.5-Math-1.5B) 모델을 기반으로 하여 독자들이 처음에는 수학 작업에서 데이터 증류의 전체 프로세스를 시도하도록 유도합니다.

## 실험 목적
- 수학적으로 관련된 데이터 세트의 정리 및 전처리 이해
- 대형 모델을 미세 조정하는 기본 방법을 이해합니다.
- 추론을 위해 대형 모델을 사용하는 기본 방법을 이해합니다.
- 모델의 수학적 능력을 평가하는 기본 방법을 이해합니다.

## 환경 요구사항

-파이썬 3.8+
-파이토치
- 트랜스포머
-데이터세트
-vllm
- 최소 40GB의 비디오 메모리를 갖춘 GPU

## 실험 문서

1. Jupyter 노트북을 엽니다.
```bash
jupyter notebook sft_math.ipynb
```
2. 노트북에는 여러 주요 부분이 포함되어 있습니다.
   -데이터세트 다운로드 및 전처리
   - 모델 로딩 및 훈련
   - 모델 생성 및 평가

3. 특정 작업 및 지침은 노트북을 참조하십시오.

## 출력 파일

훈련 프로세스는 훈련 체크포인트 파일을 생성하며, 생성된 결과는 생성이 완료된 후 저장됩니다.

## 메모

- 원본 모델 파일, 데이터 파일, 학습 후 체크포인트를 저장할 수 있는 디스크 공간이 충분한지 확인하세요. (최소 50GB 공간을 예약하세요.)
